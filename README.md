# 控制台程序操作类

# ConsoleProgramOperationClass

## 简介

可以操作控制台程序的C++类，使用C++17标准

只适用于Windows平台！

## ConsoleProgram_Sync

同步版本的控制台程序操作类，分为使用GBK等多字节字符集和使用unicode字符集(这里特指Windows API的UTF-16，即wchar_t)两种版本

多字节字符集的为``ConsoleProgram_SyncA``，unicode字符集的为``ConsoleProgram_SyncW``
